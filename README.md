# React Personnal Resume :

Run Project
1. Clone the project
```shell
git clone git@github.com:Lucas-Pleck/react-personnal-resume.git
```

2. Run the project
```shell
cd react-personnal-resume
npm install
or
yarn 
```

3. Dev
```shell
npm run dev
yarn dev
```

3. Build
```shell
npm run build
yarn build
```

4. Check
```shell
http://127.0.0.1:3000/
```
